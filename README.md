## Features

- Resource Manager
- Icon Editor
- Title Editor
- Image Editor
- Code Editor
- Manifest Editor
- Permission Editor
- Android Explorer
- Signature Viewer
- APK Cloner
- APK Signer
- APK Optimizer
- APK Installer
